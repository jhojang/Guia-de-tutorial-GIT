Guia

1. Descargar el programa git

Descargar para OSX
Descargar para windows
Descargar para Linux

2. Crea un repositorio nuevo git

Crear un a carpeta nueva y dentro de esta ejecuta

Git init


para crear un repositorio nuevo

3. Hacer checkout a un nuevo repositorio

Crea una copia local del repositorio ejecutando

git clone /path/to/repositorio


Si utilizas un servidor remoto, ejecuta

git clone username@host:/path/to/repositorio

4. add y commit

Puedes registrar cambios (añadirlos al Index) usando

git add


git add .


Este es el primer paso en el flujo de trabajo básico. Para hacer commit a estos cambios usa

git commit -m "Commit message"


Ahora el archivo esta incluído en el HEAD, pero aún no en tu repositorio remoto

5. envío de cambios

Tus cambios están ahora en el HEAD de tu copia local. Para enviar estos cambios a tu repositorio remoto ejecuta

git push origin master

git add .


Reemplaza master por la rama a la que quieres enviar tus cambios.
Si no has clonado un repositorio ya existente y quieres conectar tu
repositorio local a un repositorio remoto, usa

git remote add origin


Ahora podrás subir tus cambios al repositorio remoto seleccionado.
